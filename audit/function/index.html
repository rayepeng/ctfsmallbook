    php常用函数¶

    call_user_func_array¶

    <?php
function foobar($arg, $arg2) {
    echo __FUNCTION__, " got $arg and $arg2\n";
}
class foo {
    function bar($arg, $arg2) {
        echo __METHOD__, " got $arg and $arg2\n";
    }
}


// Call the foobar() function with 2 arguments
call_user_func_array("foobar", array("one", "two"));

// Call the $foo->bar() method with 2 arguments
$foo = new foo;
call_user_func_array(array($foo, "bar"), array("three", "four"));
?>

    使用命名空间的情况：

    <?php

namespace Foobar;

class Foo {
    static public function test($name) {
        print "Hello {$name}!\n";
    }
}

// As of PHP 5.3.0
call_user_func_array(__NAMESPACE__ .'\Foo::test', array('Hannes'));

// As of PHP 5.3.0
call_user_func_array(array(__NAMESPACE__ .'\Foo', 'test'), array('Philip'));

?>

    How could I call the constructor of a class with call_user_func_array

    下面这种方式不可能：

    $obj = new $class();
call_user_func_array(array($obj, '__construct'), $args);

    You can use reflection like:

    $reflect  = new ReflectionClass($class);
$instance = $reflect->newInstanceArgs($args);

    As of PHP 5.6.0, the ... operator can also be used for this purpose.

    if(version_compare(PHP_VERSION, '5.6.0', '>=')){
    $instance = new $class(...$args);
} else {
    $reflect  = new ReflectionClass($class);
    $instance = $reflect->newInstanceArgs($args);
}

    is_file or file_exists¶

    is_file() will return false if the given path points to a directory. file_exists()will return true if the given path points to a valid file or directory. So it would depend entirely on your needs. If you want to know specifically if it's a file or not, use is_file(). Otherwise, use file_exists().

    is_file不会解析目录，也不会解析符号链接

    file_put_contents¶

    会覆盖文件之前的内容，并且不会创建目录

    $month  = date('Yd')
file_put_contents("upload/promotions/".$month."/".$image, $contents_data); //报错！！No such file or directory

    解决：

    if (!is_dir('upload/promotions/' . $month)) {
  // dir doesn't exist, make it
  mkdir('upload/promotions/' . $month);
}

file_put_contents('upload/promotions/' . $month . '/' . $image, $contents_data);

    glob¶

    寻找与模式匹配的文件路径

    举例

    <?php
foreach (glob("*.txt") as $filename) {
    echo "$filename size " . filesize($filename) . "\n";
}
?>

    用glob函数替代opendir

    touch¶

    设定文件的访问和修改时间

    touch ( stringfilename[, inttime= time() [, int$atime]] ) : bool

    尝试将由 filename 给出的文件的访问和修改时间设定为给出的 time。 注意访问时间总是会被修改的，不论有几个参数。

    如果文件不存在，则会被创建

    举例：

    <?php
if (touch($filename)) {
    echo $filename . ' modification time has been changed to present time';
} else {
    echo 'Sorry, could not change modification time of ' . $filename;
}

// This is the touch time, we'll set it to one hour in the past.
$time = time() - 3600;

// Touch the file
if (!touch('some_file.txt', $time)) {
    echo 'Whoops, something went wrong...';
} else {
    echo 'Touched file with success';
}
?>
