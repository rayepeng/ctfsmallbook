    <?php
   $url = $_GET['url'];
   echo "Argument: ".$url."\n";
   if(filter_var($url, FILTER_VALIDATE_URL)) {
      $r = parse_url($url);
      var_dump($r);
      if(preg_match('/skysec\.top$/', $r['host'])) 
      {
         exec('curl -v -s "'.$r['host'].'"', $a);
      } else {
         echo "Error: Host not allowed";
      }
   } else {
      echo "Error: Invalid URL";
   }
?>

    1.filter_var对url进行check 2.parse_url获取url的host 3.对host进行正则匹配，判断是否以skysec.top结尾

http://localhost/web/test/22.php?url=0://evil.com:23333;skysec.top:80/

    当使用了exec()函数,例如样题代码中exec('curl -v -s "'.$r['host'].'"', $a);时 可以使用

    这样都可以绕过检测，达到请求任意ip，任意端口的目的 不妨测试：
